# Privacy Policy for _gitinder

Effective Date: April 2026

_gitinder does not collect, store, or share any personal user data.

## Authentication

_gitinder uses GitHub OAuth for user authentication. All authentication is handled securely by GitHub. The app does not have access to or store your GitHub password.

## Data Usage

The app accesses publicly available GitHub data such as:
- Repositories
- Stars
- Basic profile information

This data is used only to provide the core functionality of the app.

## Data Collection

_gitinder does not collect, store, or track personal user data.

## Third-Party Services

_gitinder relies on GitHub’s API to fetch public repository data. Please refer to GitHub’s Privacy Policy for more information:
https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement

## Data Sharing

No personal data is sold, shared, or transferred to third parties.
